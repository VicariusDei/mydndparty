# MyDndParty

Repository per la riscrittura del vecchio progetto PHP di gestione party D&D.

## Struttura iniziale

- `legacy/`: vecchio progetto estratto e sanificato, solo come riferimento funzionale.
- `docs/legacy-analysis.md`: prima mappa delle logiche e delle meccaniche recuperabili.

## Principio di lavoro

Il codice legacy non va evoluto direttamente: verrà usato per ricostruire l'app con architettura più ordinata, sicurezza migliore, modello dati più chiaro e nuove funzionalità.
